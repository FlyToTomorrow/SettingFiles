Welcome to the Outside In VW sdk.

The contents of this sdk are as follows:

redist\*               Redistributable files
sdk\common\*           Header files
sdk\resource\*         Recompilable resources
sdk\samplecode\*       Sample code
sdk\samplefiles\*      Sample files
sdk\demo\*             Compiled sample apps and configuration files.
sdk\lib\*              Compiled static libraries.
makedemo.bat           Script to copy redistributable files to sdk\demo so that samples can be run.

Documentation for this release is available online at http://docs.oracle.com/outsidein/853/index.html
